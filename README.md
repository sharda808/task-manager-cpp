# Simple Task Manager (c++)

## Features
 - Add / View / Delete Tasks
 - Mark Completed
 - File Handling (persistent Storage)
 - Search & Sort
 - OOP based design

   ## Tech Stack
   -- C++
   - STL
   - File Handling

  ## How to run
  - g++ main.cpp task.cpp file_manager.cpp -o taskManager
    ./taskManager
